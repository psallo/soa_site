# 거래명세서 생성기 (SOA Site)

간단한 입력으로 거래명세서를 작성하고, 미리보기/인쇄/PDF 다운로드까지 한 번에 처리할 수 있는 웹 기반 도구입니다.

## 주요 기능
- 공급자/공급받는자 정보 입력
- 품목 추가/삭제 및 자동 합계 계산
- 품목별 비과세 처리(세액 자동 제외)
- 미리보기(고정 출력 형태)와 인쇄 최적화(A4)
- PDF 다운로드
- 직인(도장) 이미지 업로드 및 미리보기

## 사용 방법
1. 상단 폼에 공급자/공급받는자 정보를 입력합니다.
2. 품목을 추가하고 수량/단가를 입력합니다.
3. 비과세 품목은 체크박스를 선택합니다.
4. 거래조건을 입력합니다.
5. **거래명세서 생성** 버튼으로 미리보기를 갱신합니다.
6. **인쇄** 또는 **PDF로 다운로드** 버튼을 사용합니다.

## 미리보기 안내
- 미리보기는 출력 형태에 맞춘 고정 레이아웃입니다.
- 수량/단가/공급가액/세액은 우측 정렬 및 천단위 콤마가 적용됩니다.
- 비과세 품목은 미리보기의 비고 열에 표시됩니다.

## 파일 구조
- `index.html` : 화면 구조 및 입력 폼
- `style.css` : 기본 스타일
- `print.css` : 인쇄 전용 스타일(A4)
- `script.js` : 계산 로직/미리보기 갱신/PDF 생성

## 로컬 실행
브라우저에서 `index.html`을 바로 열어 사용하거나, 간단한 정적 서버로 실행할 수 있습니다.

예시:
```
python -m http.server 8080
```
