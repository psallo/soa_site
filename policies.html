개인정보 처리방침

SOA Site는 입력 정보를 서버로 전송하지 않으며, 브라우저 저장소에만 보관합니다. 사용자가 저장 기능을 이용하지 않을 경우, 브라우저를 닫으면 입력 내용은 유지되지 않습니다.

  • 수집 항목: 사용자가 직접 입력한 문서 정보 및 선택적으로 저장한 이메일
  • 보관 위치: 사용자의 브라우저 저장소(Local Storage)
  • 보관 기간: 사용자가 직접 삭제하거나 브라우저 데이터를 삭제할 때까지
  • 제3자 제공: 없음

이용약관

본 서비스는 무료 참고용 도구입니다. 생성된 문서의 법적 효력, 세무/회계 검증 책임은 사용자에게 있습니다.

  • 서비스는 예고 없이 개선/변경될 수 있습니다.
  • 사용자는 최신 법령 및 내부 규정을 확인해야 합니다.
  • 서비스 이용으로 발생한 손해에 대해 운영자는 책임을 지지 않습니다.

광고 및 트래픽 운영 기준

광고 노출은 사용자가 실제로 콘텐츠를 탐색할 때만 이루어져야 합니다. 클릭 유도, 자동 리디렉션, 팝업 남용 등은 금지합니다.

금지되는 트래픽

  • 클릭 보상/유도형 프로그램 참여
  • 스팸 메일/메시지 유입
  • 자동 실행되는 앱/확장 프로그램 유입

권장되는 운영

  • 직접 검색 또는 북마크로 유입되는 사용성 중심 트래픽
  • 콘텐츠 중심의 안내 페이지 제공
  • 과도한 팝업, 다이얼러, 강제 리디렉션 제거

광고 표기

  • 제휴 링크는 명확히 표기
  • 스폰서드 링크에는 rel 속성 표시
  • 광고와 콘텐츠가 혼동되지 않도록 분리
